# SummGo

## Development Environment Setup

Ensure `eslint` and `eslint-plugin-react` are installed globally

```bash
npm i -g eslint eslint-plugin-react
```

## Running

```bash
npm start
```

## Running Tests

To run unit tests (using jest)

```bash
npm test
```

To run end to end tests (using cypress client)

```bash
npm run cy-open
```
